## 🚀 Quick Start (3 Minutes!)

### Step 1: Get Your API Keys
- **Telegram Token**: Message @BotFather on Telegram, send `/newbot`
- **Gemini API Key**: Visit https://aistudio.google.com/app/apikey

### Step 2: Configure Bot
- Edit `ca_bot.py` lines 18-19
- Replace `TELEGRAM_TOKEN` and `GEMINI_API_KEY` with your actual keys

### Step 3: Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 4: Run Bot
```bash
python ca_bot.py
```

### Step 5: Test in Telegram
- Search for your bot
- Send `/start`
- Start taking mock tests!
